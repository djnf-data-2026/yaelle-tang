## Dow Jones News Fund Data Journalism and Storytelling Training 2026

During the week, you will work on a project called Investigate Your Community to produce detailed story pitches relevant to your summer internship positions.

The goal is to use this week to prepare you with a few story pitches so you can work immediately at your internship.

For example, a student working at the Baltimore Sun would profile Under Armour Inc. and McCormick & Company Inc., two of the city’s major businesses, and produce a data-driven story pitch that includes a profile of the broader city and its diversity.

Investigate Your Community will equip students to:

Produce detailed story proposals
* Use data to find stories 
* Create data visualizations 
* Present data and multimedia on a simple website 
* Utilize basic investigative reporting techniques on the beat 
* Understand the value of diverse communities and information sources

### What is GitHub?
GitHub is a software that allows multiple people to work on a single coding project at the same time. It is version control software that uses a system called Git, which allows people who write code so that old versions of the files can be restored. Git can be interacted with via the command line or the GitHub Desktop application, which is much easier to deal with.

Some GitHub terms
The file you are reading now is stored in your repository. A **repository** is a group of files and folders that have been connected to Git. When you edit these files on your computer, you are working in your **local repository**. When you view these files online, you are interacting with the **remote repository**, which is a copy of the files on your computer stored in GitHub's servers. The process of getting a **remote repository** onto your computer so you can interact with the files directly is called **cloning** the repository. 


### Cloning Your Repository
1. Go to the home page of your repository and in the green Code drop down, copy the URL.
2. Open GitHub desktop, and go to file -> Clone Repsitory -> URL and paste the URL into the "URL or username/repository" textbox
3. In the local path option, you should find a location on your computer for your repository to live. Name the folder the same as your repository Anytime this folder location is changed (files added and/or deleted), these changes will be reflected to the repository.
4. Click Clone.
